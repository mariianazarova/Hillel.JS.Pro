Exercise 1

Return the century of the input year.
The input will always be a 1-4 digit string,so there is no need for validation.
[1-100] centuries.

Exercise 2

Write a function that takes a string and outputs a strings of 1's and 0's
where vowels become 1's and non-vowels become 0's.
All non-vowels including non alpha characters (spaces,commas etc.)
be included.

Exercise 3

Write a function that takes in a string of one or more words, and returns
the same string, but with all five or more letter words reversed. Strings passed in
will consist of only letters and spaces.
Spaces will be included only when more than one word is present.

Exercise 4

You are given a string of space separated numbers,
and have to return the highest and lowest number.

Exercise 5

Write a function that accepts an array of 10 integers
(between 0 and 9), that returns a string of those numbers in the form
of a phone number.
